    Description

    Base class for geometric mesh data. It stores vertex positions, normals, and UV coordinates, along with optional tangent and bitangent vectors for normal mapping.

    calculateNormals

    • calculateNormals(): void

      Calculates smooth normals and generates indices for the mesh based on its vertices. 📐 If this.indices is not already set, it generates sequential indices assuming the vertices are ordered to form triangles (e.g., 3 vertices per triangle). It computes face normals and then averages them for vertex normals to create a smooth appearance. The calculated normals and indices are stored directly in this.normals and this.indices.
